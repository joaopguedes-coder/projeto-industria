# Projeto Indústria

## Descrição

API REST desenvolvida em Node.js utilizando Express para gerenciamento de produtos industriais.

## Funcionalidades

- Cadastrar produtos
- Listar produtos
- Buscar produto por ID
- Buscar produto por nome
- Atualizar produto
- Excluir produto
- Estatísticas

## Tecnologias

- Node.js
- Express
- JSON (persistência de dados)

## Como executar

npm install

npm start

Servidor:

http://localhost:3000
